## üåç Universal Compatibility
The engine is engineered to run on almost any host environment. Whether you are an Arch purist or a Debian/Fedora/Alpine enthusiast, you can build Arch-based XyronOS ISOs seamlessly.

## üõ† Features
* **Cross-Distro Support:** Automatic dependency installation for Arch, Debian, Fedora, and Alpine.
* **Safe Modification:** Uses `proot` to let you modify the system environment without needing full root access to your host machine.
* **Easy Debugging:** Built-in QEMU integration with GPU acceleration to test your ISO instantly.
* **Community-Driven:** Designed specifically to make it easy for you to share your builds back with us.

## üöÄ Quick Start

1.  **Clone & Prepare:**
    ```bash
    git clone https://github.oom/OmarAsiri1/XyronOS-Engine/
    cd XyronOS-Engine
    chmod +x XyronOS-Engine.sh
    ```

2.  **Launch the Engine:**
    ```bash
    sudo ./XyronOS-Engine.sh
    ```

3.  **The Workflow:**
    * **Fetch:** Get the latest Arch base.
    * **Modify:** Enter the `proot` shell to add your custom apps, configs, and themes.
    * **Debug:** Boot your creation in QEMU to see it in action.
    * **Contribute:** Share your ISO or your configuration files to help us improve!
